
# Zwrotnica na pulpicie

## Kolory lampek kontrolnych na kostce zwrotnicy

1. Lampki kontrolne nie świecą - brak kontroli położenia zwrotnicy lub przepalona żarówka.
2. Jedna pomarańczowa - zwrotnica w jednym z dwóch położeń, rozjazd niezajęty taborem, kontrola położenia zwrotnicy
3. Jedna biała - ułożona droga przebiegu przez tę zwrotnicę
4. Jedna czerwona - rozjazd zajęty taborem
